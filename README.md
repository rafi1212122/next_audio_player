## Requirements
 - Node.js
 - S3 Storage
 - reCAPTCHA
 - Redis
 - PostgreSQL / MySQL (can be set in prisma)

## Getting Started

1. Create .env from .env.example

2. Install dependencies and run the development server
```bash
npm install

npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
